Help Fund Smooth UI

There are a lot of great tools out there for creating UI mockups and wireframes, but these applications are designed for a mouse and keyboard. With the explosion in popularity of tablets, hybrid laptops, and touch monitors there is a need for applications that make touch a first-class citizen. UI mockups are a perfect fit for this move to touch. We can give UI designers the ability to quickly create mockups on their touch device just like they would in a sketchbook, with all of the power and flexibility of a feature-rich web application.

So what are the features you ask? Here is what is currently in the works:

  • Saving and sharing of projects
  • Test your designs with transitions between pages
  • A library of ready to use templates, components, and icons
